# Car Marketplace
RESTful Web Service developed using Java EE technologies and Layered Architecture.

## Technology stack:
- Gradle
- JDBC
- PostgreSQL
- Servlets
- Hibernate
- Slf4j
- SQL
- JPQL
- JUnit
- Lombok
- Jackson
- HikariCP

The project hasn't finished yet.
It is in stage of refactoring the Java EE stack to Hibernate and Spring Framework.
